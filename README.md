# A.css
CSS Animation library.

## Usage
````html
<head>
  <link rel="stylesheet" href="A.css">
</head>
````

````html
<div data-A="@slide-x-in">
  <div>A</div>
</div>
````
### Mixed
````html
<div data-A="@slide-x-in @slide-y-in">
  <div>A</div>
</div>
````
